# 📚 에코시스템 문서 (Ecosystem Documentation)

**Rhee Creative**의 기술 디렉토리에 오신 것을 환영합니다. 이 문서는 **Rheehose**가 유지 관리하는 다양한 프로젝트와 라이브러리에 대한 개요를 제공합니다.

## 🏗️ 시스템 아키텍처 철학

우리의 시스템은 세 가지 핵심 기둥 위에 구축됩니다:
1. **성능 우선 (Performance First)**: 비용 없는 추상화와 효율적인 리소스 관리.
2. **시각적 서사 (Visual Narrative)**: 모든 인터페이스는 프루티거 에어로와 글래스모피즘을 통해 이야기를 전달합니다.
3. **로컬 우선 (Local-First)**: 사용자 개인정보 보호와 오프라인 기능을 우선시합니다.

## 🛠️ 애플리케이션 및 도구

### [JSONOTE](https://github.com/hslcrb/jsonote)
- **설명**: JSON을 핵심 상태로 사용하는 차세대 노트 앱.
- **기술**: Next.js 14, TypeScript, Tailwind CSS, MCP (Model Context Protocol).
- **주요 특징**: 글래스모피즘 UI, 실시간 동기화, 로컬 우선 아키텍처.
- **아키텍처**: JSON 지속성을 갖춘 Flux 영감 상태 관리.

### [Clipsmith](https://github.com/hslcrb/clipsmith-a_cpp_program)
- **설명**: 프루티거 에어로 디자인이 적용된 스마트 클립보드 매니저.
- **기술**: C++, Qt6, CMake.
- **주요 특징**: 히스토리 관리, 이미지 클리핑, 매끄러운 애니메이션.
- **내부 구조**: 크로스 플랫폼 시스템 후크를 활용한 이벤트 기반 아키텍처.

### [Sleek Downloader](https://github.com/hslcrb/pyflask_sleek-ytdownloader)
- **설명**: yt-dlp를 위한 세련된 UI 래퍼.
- **기술**: Python, Flask, CSS (Glassmorphism), Docker.
