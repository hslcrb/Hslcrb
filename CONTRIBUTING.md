# Contributing to Rhee Creative Projects

Welcome! We are excited that you are interested in contributing to our projects. Rhee Creative is dedicated to building high-quality, aesthetically pleasing software.

## How to Contribute

1.  **Find an Issue**: Look for open issues or suggest a new feature.
2.  **Fork & Clone**: Fork the repository and clone it to your local machine.
3.  **Create a Branch**: Use a descriptive branch name (e.g., `feat/new-ui-theme`).
4.  **Implement & Test**: Write your code and ensure it meets our aesthetic standards (Glassmorphism, Frutiger Aero where applicable).
5.  **Submit a PR**: Provide a clear description of your changes and why they are necessary.

## Aesthetic Guidelines

-   **Glassmorphism**: Use translucent backgrounds with backdrop-filters.
-   **Frutiger Aero**: Use glossy textures, vibrant colors, and organic shapes.
-   **Minimalism**: Keep interfaces clean and purposeful.

## Code of Conduct

Please be respectful and professional in all interactions.
