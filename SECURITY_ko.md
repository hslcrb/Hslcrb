# 보안 정책 (Security Policy)

## 지원되는 버전

저희는 주요 프로젝트의 최신 메이저 버전을 적극적으로 유지 관리합니다.

| 버전    | 지원 여부          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## 취약점 보고

보안 취약점을 발견하신 경우, [rheehose@rheehose.com](mailto:rheehose@rheehose.com)으로 이메일을 보내주세요. 48시간 이내에 보고를 확인하고, 필요한 경우 수정 일정을 제공해 드리겠습니다.

취약점이 해결될 때까지 공개적으로 공개하지 말아 주시길 부탁드립니다.
